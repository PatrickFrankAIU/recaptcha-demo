reCAPTCHA Demonstration

This demo shows how to implement Google reCAPTCHA v2 using demo site keys.

Important Notes:

  • This demo shows how reCAPTCHA protects forms from spam and abuse
  • The captcha must be completed before form submission is allowed
  • In a real application, the secret key would be used on the server to verify the token
  • Get your own keys from the Google reCAPTCHA Admin Console
